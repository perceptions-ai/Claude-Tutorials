## 拼车与共享

由于 API 成本较高，许多开发者选择合租或使用中转服务。

!!! warning "风险提示"
    使用第三方拼车或中转服务存在 API Key 泄漏或服务不稳定的风险。请务必选择信誉良好的提供商（不做推荐）。
