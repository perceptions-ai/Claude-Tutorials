# 工作流配置

打造属于你的 Vibe Coding 终极工作流。

## 什么是 "Vibe Coding"？

Vibe Coding 是一种强调流畅、即兴和人机协作的编程状态。不仅仅是写代码，更是与 AI 一起 "jamming"。

## 推荐技术栈

*   **终端**: Warp 或 iTerm2 (Mac), Windows Terminal (Windows)
*   **Shell**: Zsh + Oh My Zsh
*   **Editor**: VS Code 或 Neovim
*   **AI**: Claude Code

## 自动化脚本示例

创建一个 shell alias 快速进入 "Vibe Mode":

```bash
alias vibe="claude --temperature 0.7 --system 'You are a creative coding partner...'"
```
