# 实用技巧

提升效率的必备技巧。

## 小技巧 Checklist

- [ ] **使用 `--diff` 模式**: 查看修改前后的差异，而不是直接覆盖文件。
- [ ] **自定义 System Prompt**: 在配置中预设你的编码偏好（例如 "总是使用箭头函数"）。
- [ ] **利用 Git**: 在让 Claude 大规模重构前，务必 commit 当前状态。
- [ ] **快捷键绑定**: 在 IDE 中为常用的 Claude 指令绑定快捷键。

## 告别 Cursor？

很多用户从 Cursor 迁移到 Claude Code CLI。主要区别在于：
*   Claude Code CLI 更专注于终端体验，适合 Vim/Neovim 用户或喜欢命令行的开发者。
*   它往往能通过 Shell 直接执行命令（需授权），这比仅在编辑器内补全更具动能。
